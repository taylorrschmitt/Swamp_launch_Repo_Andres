This is the Swamp Launch IREC Payload repository for 2025-2026!
To run this, you are going to need:
1. Ardunio 2.X.X installed: https://github.com/arduino/arduino-ide/releases (this one is 2.3.8)
2. Platform IO installed (extension in VS code)
3. Raspberry Pi Pico Porject installed (entension in VS code)
4. SdFat by Bill Greiman: (install through PlatformIO in VS Code)
   a. Click the PlatformIO alien head icon in the left sidebar.
   b. Open Libraries.
   c. Search for SdFat.
   d. Find SdFat by Bill Greiman.
   e. Click Add to Project.
